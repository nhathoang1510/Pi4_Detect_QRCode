# Project RASPBERRY PI 4 DETECT QR CODE 

## Hardware Components 
1. [RASPBERRY PI 4](https://hshop.vn/products/combo-raspberry-pi-4-model-b-chinh-hang)
2. [DISPLAY HDMI 7 INCH](https://hshop.vn/products/man-honh-lcd-7-inch-hdmi-camung-dien-dung)
3. [WIDE CAMERA](https://hshop.vn/products/camera-raspberry-pi-fisheye-lens-160-degree-fov-ov5647-5mp)
## Software Components
1. [RASPIAN OS](https://www.raspberrypi.com/software/operating-systems/)
2. [OPENCV](https://opencv.org/)
3. [LIBRARY DECODE QR CODE - PYZBAR](https://github.com/NaturalHistoryMuseum/pyzbar)
## Features of Project
- Detect QR Code using Pi 4 and send data decode to realtime database.
